Brewfox

After university, I started working on game development. Brewfox is a wee 2D mobile game featuring a lot of foxes making beer. With no experience in game development, it's a new adventure where I learn and improve every day.

Designing the characters

The first step was brainstorming ideas to define the game mechanics and the style of the game. Then, I did a bit of research on designing assets for isometric games, I experimented a bit with tiles, and started designing the characters. I came up with various sketches, and picked the best one with my teammate.

Creating assets

When the foxes were ready, I created some wireframes to understand which elements I need to create for the first release. Then, I started working on tiles and game assets, trying various designs until I find the best one for the game.

Designing the UI

Now I'm working on the UI of the game, trying to create stats and icons. There are so many things that I will have to learn, but I love working on new stuff (I can't wait to get into animation), and it's a bunch of fun.
